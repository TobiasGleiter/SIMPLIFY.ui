Introduction

Copy and paste beautifully designed, lightweight web components directly into your apps. No dependencies—just HTML, CSS, and JavaScript.

This collection of reusable web components is ready to be added to your project.

No installations are required; simply use the plain HTML, CSS, and JavaScript code.

Select the components you need and integrate them into your project easily.

We’ll be adding more components over time, all free to use.

FAQ

Why copy and paste?
Copying allows you to select only the components you need, adding them as necessary. This gives you full flexibility and the ability to modify components if required.
Why no dependencies?

Dependencies can complicate your code. With pure HTML, CSS, and JavaScript, you have full control over what the components do.

Explore these components to see how simple and adaptable they are for your needs.
Which frameworks are supported?
Web components are compatible with all frameworks.
Can I use this in my project?
Yes, absolutely.
